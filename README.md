# mock-esi
Mock version of CCP ESI for testing.

# install
`docker pull antihax/mock-esi`

# run
`docker run -d -p 127.0.0.1:8080:8080 antihax/mock-esi`

# usage
Change your baseURL to http://127.0.0.1:8080 inside your tests.
